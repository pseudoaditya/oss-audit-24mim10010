Project Description
This project is centered on studying Git as an open-source software system. It covers an in-depth analysis of its origin, licensing model, ecosystem, and a comparison with proprietary alternatives. Additionally, the project includes practical implementation through Linux shell scripting to demonstrate familiarity with open-source tools and system-level operations.

The main aim of this project is to gain both theoretical knowledge and practical exposure to open-source software and understand its significance in modern computing environments.

Scripts Overview

Script 1: System Identity Report
This script provides key system details including kernel version, current user, system uptime, date, and Linux distribution.

Script 2: FOSS Package Inspector
This script verifies whether Git is installed on the system and displays relevant information such as version and description.

Script 3: Disk and Permission Auditor
This script examines important system directories and reports their disk usage along with permission details.

Script 4: Log File Analyzer
This script searches a log file for a specific keyword, counts its occurrences, and displays the matching entries.

Script 5: Open Source Manifesto Generator
This script interacts with the user to create a customized open-source manifesto based on provided inputs.

How to Run the Scripts

Step 1: Make Scripts Executable
chmod +x script1.sh
chmod +x script2.sh
chmod +x script3.sh
chmod +x script4.sh
chmod +x script5.sh

Step 2: Execute the Scripts
./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/syslog error
./script5.sh
